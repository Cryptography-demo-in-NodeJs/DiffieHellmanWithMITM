 Clone the repo

  ```bash 
  git clone https://github.com/Cryptography-demo-in-NodeJs/DiffieHellmanWithMITM.git
  ```
 Open 3 terminals first for Alice, second for Bob and third for Man in the Middle. 

  ```bash 
  cd Alice
  npm i 
  ```

  ```bash
  cd Bob 
  npm i 
  ```

  ```bash
  cd Middleman 
  npm i 
  ```
Start the servers  

  ```bash 
  node index.js
  ```
